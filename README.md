# Syntechxhub_Dashboard-UI
SellSense is a web-based sales dashboard UI that showcases key sales metrics, trends, and product insights in a clean and easy-to-understand layout. The goal of this project was to design a functional and user-friendly dashboard using common analytics patterns, while demonstrating interaction design and UX reasoning as part of an internship task.

## Description
SellSense is a web-based sales dashboard UI designed to visualize key business metrics and product performance.  
The project focuses on clear data hierarchy, usability, and common dashboard interactions used in real-world analytics products.

This project was created as part of an internship task to demonstrate UI design skills, layout structuring, and basic UX thinking.

---

## Features
- KPI cards for quick overview of sales performance  
- Sales trend charts for month-wise comparison  
- Popular products table with detailed product data  
- Separate detailed screen for extended product analysis  
- Filter dropdown to refine data by status and category  
- Month switch interaction (January and February) with updated data  
- Collapsible sidebar for better focus and usability  

---

## Design Approach
- Used a 12-column grid system for structured and responsive layout  
- Followed a clear visual hierarchy: KPIs â†’ charts â†’ tables  
- Card-based layout for better scannability and readability  
- Consistent spacing, typography, and color usage  
- Designed keeping real-world dashboard patterns in mind  

---

## UX Rationale
- Key sales metrics are placed at the top to help users understand overall performance at a glance.  
- Trend charts support comparison and analysis over time.  
- Filters and date selection allow users to change data context without leaving the dashboard.  
- Detailed product information is separated into a dedicated table view to keep the main dashboard clean and uncluttered.
